## <a name="setupdevenv"></a>設定開發環境
首先，使用 Visual Studio 和 Azure SDK 設定您的開發環境。

* Visual Studio 2017 包含 Azure SDK。 如果您使用 VS2017，不需要針對開發環境進行額外設定。
* 若為 Visual Studio 2015，請按一下下列連結以安裝 [Azure SDK for Visual Studio 2015](http://go.microsoft.com/fwlink/?linkid=518003)。
* 若為 Visual Studio 2013，按一下下列連結以安裝 [Azure SDK for Visual Studio 2013](http://go.microsoft.com/fwlink/?LinkID=324322)。

* 如果您尚未安裝 Visual Studio，請使用下列來安裝 [Visual Studio 2017](https://www.visualstudio.com/) 與 Azure SDK。

> [!NOTE]
> 視您的電腦上已有的 SDK 相依性數目而定，安裝 SDK 可能需要很長的時間 (從數分鐘到半小時以上不等)。
