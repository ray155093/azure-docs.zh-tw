# <a name="enable-the-preview-experience-to-administer-your-managed-domain-from-the-azure-portal"></a>啟用預覽體驗，從 Azure 入口網站管理受管理的網域

## <a name="your-existing-managed-domain-is-not-shown-in-the-azure-portal-preview-experience"></a>現有受管理的網域不會顯示在 Azure 入口網站 (預覽) 體驗中
您有使用 Azure 傳統入口網站建立的現有 [Azure AD Domain Services](active-directory-ds-overview.md) 受管理的網域。 當您瀏覽至 Azure 入口網站時，Azure AD Domain Services 刀鋒視窗 (預覽) 不會列出此現有受管理的網域。

![現有受管理的網域不會顯示在 Azure 入口網站中](./media/getting-started/missing-domain-services-instance.png)

這是因為現有受管理的網域尚未移轉至新的 Azure 入口網站。 您可以連絡我們以啟用現有受管理的網域之新的 Azure 入口網站 (預覽) 體驗。


## <a name="join-the-preview-for-the-azure-portal-experience-for-azure-ad-domain-services"></a>加入 Azure AD Domain Services 之 Azure 入口網站體驗的預覽
按照[與我們連絡](active-directory-ds-contact-us.md)分頁上的指示，傳送電子郵件給我們。 我們很高興能夠為您現有受管理的網域啟用 Azure 入口網站預覽。

> [!NOTE]
> Azure AD Domain Services 之新的 Azure 入口網站體驗上市 (GA) 時，所有現有受管理的網域會自動移轉。 屆時，您可以從 Azure 入口網站管理現有受管理的網域。
