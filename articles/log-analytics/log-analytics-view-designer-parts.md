# <a name="log-analytics-view-designer-visualization-part-reference"></a>Log Analytics 檢視設計工具視覺效果部分參考
Log Analytics 中的檢視設計工具可讓您在 OMS 主控台中建立自訂檢視，其中包含 OMS 存放庫中不同資料的視覺效果。 本文針對可用於自訂檢視的每個視覺效果部分提供其設定參考。

其他與檢視設計工具相關的文章︰

* [檢視設計工具](log-analytics-view-designer.md) -檢視設計工具的概觀以及建立和編輯自訂檢視的程序。
* [圖格參考](log-analytics-view-designer-tiles.md) - 可用於自訂檢視之每個圖格的設定參考。 

下表說明檢視設計工具中可用的各種圖格類型。  下列各節詳細說明每種圖格類型及其屬性。

| 檢視類型 | 說明 |
|:--- |:--- |
| [查詢清單](#list-of-queries-part) |顯示記錄檔搜尋查詢清單。  使用者可以按一下每個查詢，以顯示其結果。 |
| [數字與清單](#number-amp-list-part) |標頭中有一個數字可顯示記錄檔搜尋查詢中的記錄計數。  清單會顯示查詢的前&10; 筆結果，並包含表示數值資料行的相對值或一段時間內之變更的圖表。 |
| [兩個數字與清單](#two-numbers-amp-list-part) |標頭中有兩個數字可顯示不同記錄檔搜尋查詢中的記錄計數。  清單會顯示查詢的前&10; 筆結果，並包含表示數值資料行的相對值或一段時間內之變更的圖表。 |
| [環圈與清單](#donut-amp-list-part) |標頭會顯示從記錄檔查詢中的值資料行摘錄而來的單一數字。  環圈會以圖形方式顯示前三筆記錄的結果。 |
| [兩個時間軸與清單](#two-timelines-amp-list-part) |標頭會將一段時間內兩個記錄檔查詢的結果顯示為直條圖，其圖說文字會顯示從記錄檔查詢中的值資料行摘錄而來的單一數字。  清單會顯示查詢的前&10; 筆結果，並包含表示數值資料行的相對值或一段時間內之變更的圖表。 |
| [資訊](#information-part) |標頭會顯示靜態文字和選擇性連結。  清單會顯示一或多個包含靜態文字和標題的項目。 |
| [折線圖、圖說文字與清單](#line-chart-callout-amp-list-part) |標頭會顯示一個折線圖 (其中包含一段時間內記錄檔查詢中的多個系列)，以及包含摘錄值的圖說文字。  清單會顯示查詢的前&10; 筆結果，並包含表示數值資料行的相對值或一段時間內之變更的圖表。 |
| [折線圖與清單](#line-chart-amp-list-part) |標頭會顯示一個折線圖，其中包含一段時間內記錄檔查詢中的多個系列。  清單會顯示查詢的前&10; 筆結果，並包含表示數值資料行的相對值或一段時間內之變更的圖表。 |
| [折線圖部分堆疊](#stack-of-line-charts-part) |顯示三個獨立的折線圖，其中包含一段時間內記錄檔查詢中的多個系列。 |

## <a name="list-of-queries-part"></a>查詢清單部分
顯示記錄檔搜尋查詢清單。  使用者可以按一下每個查詢，以顯示其結果。  根據預設，檢視會包含單一查詢，您可以按一下 [+ 查詢] 來新增額外的查詢。

![查詢清單檢視](media/log-analytics-view-designer/view-list-queries.png)

| 設定 | 說明 |
|:--- |:--- |
| **一般** | |
| 標題 |顯示於檢視頂端的文字。 |
| 新增群組 |選取即可在檢視中建立新群組 (開始於目前檢視)。 |
| 預先選取的篩選器 |當使用者選取查詢時，要納入左側篩選窗格中的屬性清單 (以逗號分隔)。 |
| 轉譯模式 |選取查詢時顯示的初始檢視。  使用者可以在開啟查詢後，選取任何可用的檢視。 |
| **查詢** | |
| 搜尋查詢 |要執行的查詢。 |
| 易記名稱 |要對使用者顯示的查詢描述性名稱。 |

## <a name="number--list-part"></a>數字與清單部分
標頭中有一個數字可顯示記錄檔搜尋查詢中的記錄計數。  清單會顯示查詢的前&10; 筆結果，並包含表示數值資料行的相對值或一段時間內之變更的圖表。

![查詢清單檢視](media/log-analytics-view-designer/view-number-list.png)

| 設定 | 說明 |
|:--- |:--- |
| **一般** | |
| 群組標題 |顯示於檢視頂端的文字。 |
| 新增群組 |選取即可在檢視中建立新群組 (開始於目前檢視)。 |
| 圖示 |顯示於標頭中結果旁邊的影像檔。 |
| 使用圖示 |選取即可顯示圖示。 |
| **標題** | |
| 圖例 |顯示於標頭頂端的文字。 |
| 查詢 |針對標頭執行的查詢。  將會顯示查詢所傳回的記錄計數。 |
| **清單** | |
| 查詢 |針對清單執行的查詢。  將顯示結果中前&10; 筆記錄的前兩個屬性。  第一個屬性應該是文字值，而第二個屬性是數值。  根據數值資料行的相對值，自動建立長條圖。<br><br>在查詢中使用 Sort 命令，將清單中的記錄排序。  使用者可以按一下 [檢視全部]，以執行查詢並傳回所有記錄。 |
| 隱藏圖形 |選取以停用數值資料行右側的圖形。 |
| 啟用走勢圖 |選取以顯示走勢圖 (而非水平橫條圖)。  請參閱[一般設定](#sparklines)以瞭解詳細資料。 |
| 色彩 |橫條或走勢圖的色彩。 |
| 名稱和值分隔符號 |您要將 text 屬性剖析成多個值時的單一字元分隔符號。  請參閱[一般設定](#name-value-separator)以瞭解詳細資料。 |
| 瀏覽查詢 |當使用者選取清單中項目時所要執行的查詢。  請參閱[一般設定](#navigation-query)以瞭解詳細資料。 |
| **清單** |**> 資料行標題** |
| 名稱 |顯示於清單第一欄頂端的文字。 |
| 值 |顯示於清單第二欄頂端的文字。 |
| **清單** |**> 臨界值** |
| 啟用臨界值 |選取以啟用臨界值。  請參閱[一般設定](#thresholds)以瞭解詳細資料。 |

## <a name="common-settings"></a>一般設定
以下小節說明幾個視覺化部分的通用設定。

### <a name="a-namename-value-separatorname--value-separatora"></a><a name="name-value-separator">名稱和值分隔符號</a>
您要將來自清單查詢的 text 屬性剖析成多個值時的單一字元分隔符號。  如果您指定分隔符號，您可以在 [名稱] 方塊中提供每個欄位的名稱 (以相同的分隔符號分隔)。

例如，考慮名為 *Location* 的屬性，其包含 *Redmond-Building 41* 和 *Bellevue-Building12* 等值。  您可以針對 [名稱與值分隔符號] 指定 –，針對 [名稱] 指定 City-Building。  這會將每個值剖析成兩個名為 City 和 Building 的屬性。 

### <a name="a-namenavigation-querynavigation-querya"></a><a name="navigation-query">瀏覽查詢</a>
當使用者選取清單中項目時所要執行的查詢。  使用 {selected item} 來包含使用者所選項目的語法。

例如，如果查詢具有名為 Computer 的資料行且瀏覽查詢是 {selected item}，則會在使用者選取電腦時執行 Computer="MyComputer" 之類的查詢。  如果瀏覽查詢是 *Type=Event {selected item}*，則會執行 *Type=Event Computer="MyComputer"*。

### <a name="a-namesparklinessparklinesa"></a><a name="sparklines">走勢圖</a>
走勢圖是說明一段時間內清單項目值的小型折線圖。  針對包含清單的視覺化部分，您可以選取要顯示表示數值資料行相對值的水平橫條圖，或顯示表示其一段時間內的值的走勢圖。

下表描述走勢圖的設定。

| 設定 | 說明 |
|:--- |:--- |
| 啟用走勢圖 |選取以顯示走勢圖 (而非水平橫條圖)。 |
| 作業 |如果啟用走勢圖，此即為對清單中每個屬性執行的作業，以計算走勢圖的值。<br><br>- 最後一個範例︰時間間隔內系列的最後一個值。<br>- 最大值︰時間間隔內系列的最大值。<br>- 最小值︰時間間隔內系列的最小值。<br>- 總和︰時間間隔內系列的值總和。<br>- 摘要︰使用相同的 measure 命令做為標頭中的查詢。 |

### <a name="a-namethresholdsthresholdsa"></a><a name="thresholds">臨界值</a>
臨界值可讓您在清單中每個項目的旁邊顯示彩色圖示，以針對超出特定值或落在特定範圍內的項目提供明顯的視覺指標。  例如，您可以為含有可接受值的項目顯示綠色圖示，如果值是在指示警告的範圍內就顯示黃色圖示，如果值超出錯誤值，則顯示紅色圖示。

當您針對組件啟用臨界值時，必須指定一或多個臨界值。  如果項目的值大於某個臨界值，並低於下一個臨界值，就使用該顏色。  如果項目大於最高臨界值，就會設定該顏色。   

每個臨界值組都有一個具有 **Default** 值的臨界值。  如果沒有超出其他值，就會設定這個色彩。  您可以按一下 **+** 或 **x** 按鈕來新增或移除臨界值。

下表描述臨界值的設定。

| 設定 | 說明 |
|:--- |:--- |
| 啟用臨界值 |選取一個色彩圖示以在每個值的左邊顯示，來指示和指定臨界值有關的健康狀態。 |
| 名稱 |用於識別臨界值的名稱。 |
| 閾值 |臨界值。  每個清單項目的健康狀態色彩都是設定為項目值所超出之最高臨界值的色彩。  會有一個 Default 臨界值，這是沒有超出任何臨界值情況下的色彩。 |
| 色彩 |臨界值的色彩。 |
